Verify

Verify is an Amazon Alexa voice controlled application. The goal of Alexa was to be a tool for people who need to log easily forgettable repetitive tasks. By simply telling Alexa to create a log, Alexa will create a timestamp for that moment. Users can recall their last log by making a verbal request.


Technologies

  • AWS Lambda
  • Amazon Skill Kit
  • DynamoDB
  • Node.js

This was a great project to use a completely different and new technology. Breaking out of the browser was a blast and challenge all wrapped into a fun project.
